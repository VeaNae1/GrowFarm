잎 결핍 원소 분석을 위한 모델과 어플리케이션

## Purpose
* 식물의 잎 사진을 받아 이미지 분석
* 분석 결과를 토대로 결핍 가능성 분석
* 사용자에게 필요한 대응 추천

## Key Features
* 앱에서 사진 촬영 및 갤러리 접근 후 선택
* 서버를 통해 모델로 사진 전송
* 모델에서 받아오는 결과를 DB에 저장
* 앱으로 사용자에게 제공
